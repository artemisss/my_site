# AGENTS.md

Контекст для AI-агентов и контрибьюторов, работающих с этим репозиторием.

## О проекте

**Публичный проект.** Это персональный сайт-визитка Артёма Королёва.
Исходники открыты, не содержат секретов и предназначены для публичного хостинга
(GitHub Pages / любой статический хостинг). Не добавляй сюда токены, ключи,
приватные данные или что-либо, что нельзя показывать публично.

- **Владелец:** Королёв Артём
- **Род занятий:** Product Manager в Wildberries
- **Назначение:** одностраничная визитка — имя, чем занимаюсь, кнопка с контактом
- **Контакт:** Telegram (ссылка в кнопке)

## Структура

```
my_site/
├── index.html   # весь сайт: разметка + встроенные стили (single-file)
└── AGENTS.md    # этот файл
```

Проект намеренно состоит из одного самодостаточного файла — без сборки,
зависимостей, фреймворков и внешних ресурсов. Всё (HTML, CSS, SVG-иконка)
лежит в [index.html](index.html).

## Принципы дизайна

Держись этих принципов при любых правках:

- **Минимализм в стиле Apple.** Белый фон, много воздуха, минимум текста.
- **Системный шрифт.** `-apple-system, BlinkMacSystemFont, "SF Pro Display"…` —
  никаких подключаемых веб-шрифтов.
- **Liquid glass.** Кнопки — полупрозрачное «стекло»: `backdrop-filter` (blur +
  saturate), внутренние блики, скользящий световой отблеск на hover.
- **Сдержанная анимация.** Плавные `cubic-bezier`-переходы; всё уважает
  `prefers-reduced-motion`.
- **Без зависимостей.** Не добавляй npm-пакеты, CDN-ссылки или внешние шрифты —
  сайт должен открываться двойным кликом и работать офлайн.

## Как запускать

Сборка не нужна. Открой [index.html](index.html) в браузере (двойной клик),
либо подними локальный сервер для удобства:

```bash
python3 -m http.server 8000   # затем http://localhost:8000
```
